
/api/rest/operate/device/floor/{deviceId}?type=&floorId=&floorNum=&scientist=&name=&email=&mobile=&productNum=
deviceId：当前设备Id
入参：{
type    1是修改，0是添加
floorId 修改时制定当前的层Id
floorNum    添加的第几层的数量，int值
scientist   科学家名字
name        存放物品名称
email       科学家邮箱
mobile      科学家手机
productNum  实验品数量
}

/api/rest/firm/query/company/{name}?start=&limit=&name=&address=                平台管理员和业务员企业列表中显示
{name}用户登录名
入参：{
start           默认0
limit           默认10
name            企业名称
address=        企业地址：模糊匹配
}
出参：{
total：      查询到的总数
thisNum：    本次查询到数目
companies：  企业列表
}
